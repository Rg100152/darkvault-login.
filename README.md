🔐 DarkVault

DarkVault is a modern dark-themed authentication and account management web application built with pure HTML, CSS, and JavaScript. It features a stylish glassmorphism-inspired interface with pink, maroon, red, blue, and black accents, providing a premium user experience.

✨ Features

- User Registration System
- Secure Login Interface
- CAPTCHA Verification
- Local Storage Data Management
- Responsive Mobile-Friendly Design
- Modern Glassmorphism UI
- Dashboard with Account Information
- Logout Functionality
- Account Deletion Support
- Smooth Animations and Transitions

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage API

📱 Responsive Design

DarkVault is optimized for desktop, tablet, and mobile devices, ensuring a smooth user experience across different screen sizes.

🚀 Getting Started

1. Download or clone the repository.
2. Open "index.html" in your browser.
3. Create a new account.
4. Login and explore the dashboard.

⚠️ Disclaimer

This project is intended for educational and demonstration purposes. User data is stored locally in the browser using Local Storage and should not be considered a production-ready authentication system.
